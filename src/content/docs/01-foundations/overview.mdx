<span class="editorial-label">第 1 部分 · 概览</span>

打地基的一部分。学完这一部分，你能装好 n8n、看懂它的编辑器界面，跑出第一个工作流，并且**真正理解"一条工作流到底在做什么"**。

## 这一部分会讲

1. **安装 n8n** —— Cloud（最快）/ Desktop（最简单）/ Docker（最灵活）三种方式对比
2. **编辑器界面巡礼** —— 画布、节点库、表达式编辑器
3. **你的第一个 Workflow** —— Hello World 工作流：手动触发 → Set → 完成
4. **核心心智模型** —— 什么是 Item、什么是 JSON、节点如何"一项一项处理"数据
5. **Execution 执行机制** —— 一次运行到底发生了什么、如何查看日志和调试
