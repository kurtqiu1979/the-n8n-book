<span class="editorial-label">Part 1 · Overview</span>

This is the foundation. By the end of this part you'll have installed n8n, learned the editor, shipped your first workflow, and **really understood what a workflow is doing under the hood**.

## What you'll learn

1. **Install n8n** — Cloud (fastest) / Desktop (simplest) / Docker (most flexible), compared
2. **Tour the editor** — canvas, node library, expression editor
3. **Your first workflow** — Hello World: manual trigger → Set → done
4. **The core mental model** — what an Item is, what JSON is, how nodes process data one item at a time
5. **Execution mechanics** — what happens when you hit "Execute Workflow", how to read logs and debug
