n8n ships **500+ app nodes** — Slack, Notion, OpenAI, Shopify, you name it. These "ready-made" nodes are what makes n8n more pleasant than rolling your own HTTP calls. This section gives you the lay of the land — so that **the moment you have a need, you know which node to grab**.

## App node vs HTTP Request: which?

| | App Node | HTTP Request |
|---|---|---|
| **Speed to build** | Fast (form UI) | Slow (read docs) |
| **Coverage** | Common APIs | **Every** API |
| **New feature lag** | Wait for n8n update | Use immediately |
| **Auth** | Configure once | Per-workflow |

**Heuristic:** **prefer the app node; fall back to HTTP Request when the node doesn't cover what you need.**

## Resource × Operation: the 2D selector

Every app node has two top-level dropdowns:

- **Resource** — kind of object (Notion: `database / page / block`)
- **Operation** — what to do with it (`get / create / update / delete / list / search`)

<TerminalCell filename="notion-resource-operation.txt" runTag="📦">
{`Notion's Resource × Operation matrix:

Resource: Database
  Operations: get, getMany, getDatabasePages, search

Resource: Page
  Operations: get, getMany, create, update, archive

Resource: Block
  Operations: getMany, append, update, delete

Resource: User
  Operations: get, getMany`}
</TerminalCell>

This Resource × Operation pattern repeats across 50+ app nodes — **learn one, learn a family**.

<TipCallout type="warn" title="Top trap: permission errors">
  90% of the time the cause is misconfigured **Credentials** — OAuth scope insufficient, API key missing permissions. Next section covers it.
</TipCallout>

## Can't find a node?

1. **Refine search** — n8n's search is loose; try synonyms
2. **Use HTTP Request** — fallback always works
3. **Community nodes** — self-host can install 300+ community-built nodes from npm (`n8n-nodes-*`)
4. **Custom nodes** — last resort; out of scope for this book

<TipCallout type="quest" title="Extension challenge">
  Pick a SaaS app you use daily. Open its n8n node, list its Resource × Operation combos. Compare against the app's official API docs — what % is covered?
</TipCallout>
