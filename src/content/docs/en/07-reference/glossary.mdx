Stuck on terminology while reading n8n docs? Look it up here.

## Core concepts

| English | 中文 | One-line meaning |
|---|---|---|
| Workflow | 工作流 | A set of connected nodes — the automation itself |
| Node | 节点 | The basic unit of a workflow |
| Connection | 连线 | Direction of data flow between nodes |
| Trigger | 触发器 | Entry node that starts a workflow |
| Action | 动作节点 | Node that performs a specific action |
| Execution | 执行 | One run instance of a workflow |
| Item | 数据项 | One JSON object in the data array |
| Binary | 二进制数据 | File attachment (image, PDF, etc.) |
| Credential | 凭证 | Identity info for API calls (encrypted) |
| Webhook | Webhook | HTTP endpoint that listens for requests |
| Active / Inactive | 激活 / 未激活 | Whether the workflow runs in the background |
| Pinned Data | 钉住的数据 | Locked node output for debugging |

## Expressions

| English | 中文 | Meaning |
|---|---|---|
| Expression | 表达式 | JS code in `{{ }}` |
| Expression Mode | 表达式模式 | Input field is "evaluated" |
| Fixed Value | 固定值（模式） | Input field is "literal" |
| `$json` | (current item's JSON) | Special n8n-injected variable |
| `$('NodeName')` | Reference a node | Cross-node data access |
| `.item` | Smart-paired item | Use cautiously |
| `.first()` / `.last()` / `.all()` | First / Last / All items |
| Luxon | (a JS date lib) | `$now` and friends built on it |
