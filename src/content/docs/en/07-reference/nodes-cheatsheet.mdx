<span class="editorial-label">Part 7 · Node Cheat Sheet</span>

n8n has 500+ nodes; 80% of workflows use about 60. This page lists the common ones, indexed two ways: **by purpose** and **alphabetically**.

## By purpose

### Triggers

| Node | Use |
|---|---|
| Manual Trigger | Click-to-run (debug) |
| Schedule Trigger | Time-based (Cron) |
| Webhook | Listen for HTTP requests |
| Chat Trigger | Built-in chat UI |
| Execute Sub-workflow Trigger | Called by other workflows |
| Error Trigger | Fires on workflow failure |

### App Triggers

Telegram · Slack · Gmail · GitHub · Stripe · Shopify · Notion · Airtable · Google Sheets · Google Calendar · Typeform · RSS Feed

### Core / Data ops

| Node | Use |
|---|---|
| Set / Edit Fields | Add/modify/delete/rename fields |
| IF | Binary branching |
| Switch | Multi-way routing |
| Merge | Combine two streams (5 modes) |
| Loop Over Items | Batch iteration / rate-limit |
| Split Out | Array field → multiple items |
| Aggregate | Multi-item → single |
| Sort / Filter / Limit | Self-explanatory |
| Remove Duplicates | Dedup |
| Code | Write JS / Python |
| Wait | Pause N sec/min/days |
| Stop and Error | Manually fail |

### HTTP / Communication
