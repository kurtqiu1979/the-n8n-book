> **Difficulty** 🟡 Medium · **Time** ~40 min · **Core nodes** YouTube API × AI Agent (JSON mode)

Uploaded the video but too lazy to write good SEO? This workflow scans your new uploads daily, pulls transcripts, uses AI to generate clickbait-but-honest titles, keyword-rich descriptions, tag arrays, and chapter timestamps — all auto-pushed to YouTube.

## 🎯 Scenario

- Solo YouTubers shipping 1-2 videos a day
- Tutorial channels — chapter timestamps boost completion rate
- Teams managing multiple channels

## 📋 Key configs

### YouTube - List Recent Videos

<TerminalCell filename="yt-list.txt" runTag="⚙">
{`Resource:   Video
Operation:  Search
Channel ID: <your channel ID>
Published After: ={{ $now.minus({days: 1}).toISO() }}

Only last 24 hours' uploads.`}
</TerminalCell>

### Fetch Transcript

The YouTube node doesn't expose transcripts directly. HTTP Request against YouTube Captions API:

<TerminalCell filename="fetch-transcript.txt" runTag="⚙">
{`Method:  GET
URL:     =https://www.googleapis.com/youtube/v3/captions?videoId={{ $json.id.videoId }}&part=snippet
Auth:    OAuth2 (Google credentials)`}
</TerminalCell>

### SEO Generator (AI Agent)

<TerminalCell filename="seo-ai.txt" runTag="⚙">
{`System Message:
  You are a YouTube SEO expert. After reading the transcript, generate JSON:
  {
    "title": "<60-char hook title>",
    "description": "<first 100 chars with keywords, then chapter timestamps>",
    "tags": ["<15 SEO tags>"],
    "chapters": [{"time": "00:00", "title": "Intro"}, ...]
  }
  Return ONLY JSON.

Attached OpenAI:
  Model:           gpt-4o    ← use gpt-4o (transcripts are long, need stronger model)
  Response Format: json_object`}
</TerminalCell>

### YouTube - Update Metadata

<TerminalCell filename="yt-update.txt" runTag="⚙">
{`Operation: Update Video
Video ID:  ={{ $('YouTube - List Recent Videos').item.json.id.videoId }}
Updates:
  title:        ={{ $json.title }}
  description:  ={{ $json.description }}
  tags:         ={{ $json.tags.join(',') }}`}
</TerminalCell>
