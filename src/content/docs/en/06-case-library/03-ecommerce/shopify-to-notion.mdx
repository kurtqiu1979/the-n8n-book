> **Difficulty** 🟢 Easy · **Time** ~15 min · **Core nodes** Shopify Trigger × Notion × Slack

The classic small-team setup: each new Shopify order auto-creates a Notion DB row (for fulfillment tracking, customer service), and Slack pings the team in real-time so nothing slips.

## 🎯 Scenario

- Small store: no full ERP yet, Notion as order tracker
- Team coordination: Slack ping on every order, never miss one
- Data backup: independent copy outside Shopify

## 📋 Key nodes

### Shopify Trigger

<TerminalCell filename="shopify-trigger.txt" runTag="⚙">
{`Events:  [orders/create]
Credentials: Shopify Access Token

n8n auto-registers a webhook in your Shopify admin.`}
</TerminalCell>

### Reshape Order (Set)

Shopify's webhook payload is verbose — use Set to pull out essentials:

<TerminalCell filename="reshape.txt" runTag="⚙">
{`order_id:        ={{ $json.id }}
customer_email:  ={{ $json.email }}
total:           ={{ Number($json.total_price) }}     ← string → number
items_summary:   ={{ $json.line_items.map(i => i.quantity + 'x ' + i.name).join(', ') }}`}
</TerminalCell>

### Create Notion Page

<TerminalCell filename="notion-create.txt" runTag="⚙">
{`Database:  <your Notion Orders DB>
Properties:
  Order (title):        =#{{ $json.order_id }}
  Email (email):        ={{ $json.customer_email }}
  Total (number):       ={{ $json.total }}
  Items (text):         ={{ $json.items_summary }}
  Status (select):      Pending`}
</TerminalCell>

<TipCallout type="warn" title="Trap · Notion field types must match">
  
  Notion property types (title / rich_text / number / select / email / phone / date / ...) **must match** the node parameter types. Mismatches get rejected. Set up the Notion DB schema first.
  
</TipCallout>

<TipCallout type="quest" title="Extensions">
