<span class="editorial-label">Part 6 · Overview</span>

After Parts 1–5 you know how to build workflows from scratch. This part gives you **24 wheels you can just lift** — from a simple "daily email digest" to a full-blown "AI customer support with RAG".

Every case includes:
- 🎯 **Backstory** — what it solves, who needs it
- 🔍 **Workflow diagram** — interactive (pan / zoom), embedded inline
- 📋 **Downloadable Workflow JSON** — copy-paste into your own n8n
- ⚠️ **Pitfall warnings** — which parameters most often go wrong
- 🎁 **Extension challenges** — taking it further

## 8 thematic groups

- 🤖 **AI Agents & Chatbots** (Telegram AI Bot / PDF RAG / AI Company Enrichment)
- ✍️ **Content Ops & Social** (Multi-Platform Poster / YouTube SEO / Reddit Trends)
- 🛒 **E-commerce & SaaS Sync** (Shopify → Notion / Stripe Failures / Trial Expiry)
- 📅 **Personal Productivity / GTD** (Daily Email Digest / Calendar → Notion / Habit Bot)
- 🔄 **Data Pipelines** (CSV → DB / API Polling Router / Multi-source Dedup)
- 📣 **Marketing & CRM** (Lead Enrichment / Drip Campaigns / Typeform → Salesforce)
- 🛠 **DevOps Lite** (Uptime Monitor / GitHub → Discord / Backup Audit)
- 🎁 **Fun Projects** (Weather Wallpaper / AI Birthday Reminders / Personal RSS Butler)
