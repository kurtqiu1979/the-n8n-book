> **Difficulty** 🟢 Easy · **Time** ~20 min · **Core nodes** Notion × IF × AI

Too many friends, can never remember birthdays? Maintain a Notion "Friends" DB (Name / Birthday / Notes), this workflow scans daily, when someone's birthday is today, AI uses the Notes field ("loves hiking, just changed jobs") to **compose a personalized wish**, pushed to your Telegram — just copy/paste to send.

## 📋 Key configs

<TerminalCell filename="key-config.txt" runTag="⚙">
{`Notion DB schema (create first):
  Name (title)
  Birthday (date)
  Notes (rich text) - what you know about them

List Friends:
  Database: <your Friends DB>
  Return All: true

Is Birthday Today? (IF):
  ={{ DateTime.fromISO($json.properties.Birthday.date.start).toFormat('MM-dd') }}
    ==
  ={{ $now.toFormat('MM-dd') }}

AI Compose Wish:
  Prompt: Write a birthday wish for {{ name }}, based on notes: {{ notes }}
  System: Warm, personal, no clichés. Under 100 words.

Telegram Notify Me:
  Chat ID: <your Telegram>
  Text: 🎂 Today is X's birthday + AI-composed wish`}
</TerminalCell>

<TipCallout type="warn" title="Trap · Notion date parsing">
  
  Notion's date field is `{ start: "2026-05-15", end: null }`. Always access `.date.start`, parse with `DateTime.fromISO(...)`. Don't string-compare raw — format gotchas await.
  
</TipCallout>
