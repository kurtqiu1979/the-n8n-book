## 📋 Key configs

<TerminalCell filename="key-config.txt" runTag="⚙">
{`Fetch Weather:
  URL: https://wttr.in/shanghai?format=j1
  → free public API, no key needed

AI Generate Prompt:
  System: Generate a DALL-E prompt from weather data
          Style: serene, poetic watercolor
          Output prompt only, no explanation

Generate Wallpaper (OpenAI Image):
  Model:   dall-e-3
  Prompt:  ={{ $json.output }}
  Size:    1792x1024  (16:9 desktop)

Save to Drive:
  File name: =wallpaper-{{ $now.toFormat('yyyy-MM-dd') }}.png
  Parent:    <Drive Wallpapers folder>`}
</TerminalCell>
