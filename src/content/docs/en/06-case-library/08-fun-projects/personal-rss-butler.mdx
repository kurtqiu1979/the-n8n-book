> **Difficulty** 🟡 Medium · **Time** ~25 min · **Core nodes** RSS × Aggregate × AI Curator

The classic paradox: "dozens of RSS subscriptions, but the RSS reader never gets opened." This workflow **scans daily for you**, AI categorizes by topic + picks 5 highlights with summaries, sends a Markdown digest to Telegram — **5-minute commute read of today's must-knows**.

## 📋 Key configs

<TerminalCell filename="key-config.txt" runTag="⚙">
{`Set Feed URLs:
  feeds: "https://news.ycombinator.com/rss,
          https://blog.n8n.io/feed,
          https://www.anthropic.com/news/rss.xml"

Split Feeds:
  Field: feeds, Delimiter: ","
  → one item per feed

Read RSS:
  URL: ={{ $json.feeds }}
  Ignore Older Than: 86400 (seconds) ← only last 24h

Aggregate:
  Fields: [title, link]
  → combine all feeds' items into one array

AI Curator:
  System: Categorize by topic (AI / Tech / Product / Other)
          Pick top 5 with 200-word summaries each
          Markdown output with links

Send to Telegram:
  parseMode: Markdown
  ← so links render`}
</TerminalCell>

  Long-running, some feeds will die or change URLs. **Continue On Fail on each Read RSS** — one dead feed shouldn't kill the digest.
