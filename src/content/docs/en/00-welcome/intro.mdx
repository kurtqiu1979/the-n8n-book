<span class="editorial-label">Preface · Part 0</span>

This is an n8n textbook for **non-developer automation enthusiasts**. If you've heard of Zapier or Make, or simply want repetitive things to run themselves, you've arrived. The book assumes you can't code — yet by the last page, you'll have the ability to **design a complete workflow on your own**.

## What you'll learn

- Get n8n running on your machine or a server
- Build the mental model: *node → connection → execution*, and explain why a workflow looks the way it does
- Make nodes talk to each other through expressions and data flow
- Handle branches, loops, and errors — write **production-grade** workflows
- Wire up an AI Agent that can call tools and remember context
- 24 real-world cases spanning e-commerce, content ops, CRM, AI, and personal productivity

## The book has two faces

**By day**: a printed textbook. Cream paper, serif headings, drop caps, generous whitespace. For slow, deliberate reading.

**Hands-on**: an open notebook. Dark cells, monospace, copyable JSON, pannable node graphs. For running, tweaking, experimenting.

<TipCallout type="tip" title="Recommended pace">
  Read parts 1–5 linearly. Jump around the case library (part 6) by interest. Summon the reference toolkit (part 7) anytime with `Cmd+K`.
</TipCallout>

## Finding what you need

- **Left sidebar**: full table of contents, collapsible by part
- **Right sidebar**: page outline — never lose your place in a long page
- **Top search**: `Cmd+K` (Mac) or `Ctrl+K` (Windows) for full-text search
- **Top-right**: switch between English and Chinese
- **Footer**: previous / next lesson navigation

<TipCallout type="warn" title="About versioning">
  This book targets n8n 1.x. n8n moves fast — when UI details drift, the **core concepts still apply**. Every lesson is dated at the bottom.
</TipCallout>
