读 n8n 官方英文文档时被术语卡住？这页全收。**双向查**：中文找英文 / 英文找中文都行。

## 核心概念

| 英文 | 中文 | 一句话解释 |
|---|---|---|
| Workflow | 工作流 | n8n 里一组节点连成的自动化流程 |
| Node | 节点 | workflow 的基本组成单元 |
| Connection | 连线 | 节点间的数据流向 |
| Trigger | 触发器 | 启动 workflow 的入口节点 |
| Action | 动作节点 | 执行具体操作的节点 |
| Execution | 执行 | workflow 的一次运行实例 |
| Item | 数据项 | 数据数组里的一个 JSON 对象 |
| Binary | 二进制数据 | 文件附件（图片、PDF 等） |
| Credential | 凭证 | 调用 API 的身份信息（加密存储） |
| Webhook | Webhook | 监听 HTTP 请求的端点 |
| Active / Inactive | 激活 / 未激活 | workflow 是否在后台运行 |
| Pinned Data | 钉住的数据 | 调试时锁定的节点输出 |

## 表达式

| 英文 | 中文 | 说明 |
|---|---|---|
| Expression | 表达式 | `{{ }}` 包裹的 JS 代码 |
| Expression Mode | 表达式模式 | 输入框的"求值"状态 (vs Fixed Value) |
| Fixed Value | 固定值（模式） | 输入框的"字面量"状态 |
| `$json` | (当前 item 的 JSON) | n8n 注入的特殊变量 |
| `$('NodeName')` | 引用某节点 | 跨节点取数据 |
| `.item` | 智能配对的 item | 谨慎使用 |
| `.first()` / `.last()` / `.all()` | 第一个 / 最后 / 全部 items |
| Luxon | (一个 JS 日期库) | n8n 注入的 `$now` 等基于它 |
