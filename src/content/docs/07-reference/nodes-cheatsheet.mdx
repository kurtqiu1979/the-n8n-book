<span class="editorial-label">第 7 部分 · 节点速查表</span>

n8n 有 500+ 节点，但 80% 的工作流只用其中 60 个。这一页按"用途"和"字母"双索引列出最常用节点，**遇到需求直接查表**。

## 按用途分类

### 触发器 (Triggers)

| 节点 | 用途 | 典型 case |
|---|---|---|
| Manual Trigger | 手动按按钮启动（调试） | 学习阶段所有 demo |
| Schedule Trigger | 按时间表自动跑（Cron） | 每天 9 点拉数据 |
| Webhook | 监听 HTTP 请求 | 接收 form 提交 / 三方回调 |
| Chat Trigger | 内置 chat UI 入口 | AI 聊天助手 |
| Execute Sub-workflow Trigger | 被别的 workflow 调用 | 模块化复用 |
| Error Trigger | workflow 出错时启动 | 全局错误处理 |

### 应用触发器 (App Triggers)

|---|---|
| Telegram Trigger | 监听 Telegram 消息 |

| Gmail Trigger | 收新邮件 |
| GitHub Trigger | PR / Issue / Release 等 |
| Stripe Trigger | 支付事件 |
