<span class="editorial-label">第 4 部分 · 概览</span>

新手 → 进阶玩家的分水岭。学完这一部分，你的工作流能优雅处理"如果 / 否则 / 出错怎么办 / 怎么循环 / 怎么模块化"这些真实生产场景里的硬骨头。

## 这一部分会讲

1. **IF / Switch** —— 条件分支
2. **Merge 详解** —— 合并模式，附图解（这是新手最容易翻车的地方）
3. **SplitInBatches** —— 批处理与限速（防止把第三方 API 打爆）
4. **Loop / 循环模式** —— 几种常见循环写法
5. **错误处理体系** —— Continue On Fail、Error Trigger、Error Workflow
6. **Sub-workflow 模块化** —— 让 Workflow 像函数一样复用
