如果说节点是肌肉，表达式就是神经——它让你的数据在节点之间灵活穿梭、按你的逻辑变形。

## 这一部分会讲

1. **`{{ }}` 表达式语法入门** —— 什么时候要用、怎么切换"固定值 / 表达式"两种模式
2. **引用上游数据** —— `$json` / `$('NodeName')` / `.first()` / `.all()` / `.last()`
3. **常用 JavaScript 函数速查** —— 字符串、日期、数组操作
4. **类型转换与日期处理** —— 字符串/数字/布尔/日期的互转
5. **新手必踩的 5 个坑** —— `item` 引用错、表达式 vs 固定值、`.item` 误用 等
