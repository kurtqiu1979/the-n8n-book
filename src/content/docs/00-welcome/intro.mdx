这是一本写给**零基础自动化爱好者**的 n8n 教材。如果你听说过 Zapier、Make，或者只是单纯想让重复的事情自动跑起来，那么你来对地方了。本书不假设你会写代码——但读到最后，你会拥有"自己设计一个完整工作流"的能力。

## 你将学到的

- 让 n8n 在你的电脑或服务器上跑起来
- 看懂"节点 → 连线 → 执行"这套心智模型，能解释"为什么这个 workflow 是这样的"
- 用表达式和数据流让节点之间"对话"
- 处理分支、循环、错误，写出**生产级别**的工作流
- 用 AI Agent 节点搭一个能调用工具的智能助手
- 24 个完整案例，覆盖电商、内容、CRM、AI、个人生产力等真实场景

## 这本书的两个气质

这本书有两副面孔——

**白天的它**：像一本精装教科书。米色底、衬线大字、段首下沉、留白慷慨。适合慢读、深思。

**动手的它**：像一台开着的笔记本。深色 cell、等宽字体、可复制的 JSON、可平移缩放的节点图。适合跑、改、试。

<TipCallout type="tip" title="阅读节奏建议">
  线性章节（第 1–5 部分）按顺序读；案例库（第 6 部分）按你的兴趣跳读；速查工具箱（第 7 部分）随时 `Cmd+K` 召唤。
</TipCallout>

## 怎么找到你要的东西

- **左侧栏**：完整目录，章节折叠展开
- **右侧栏**：当前页大纲，长文章不迷路
- **顶部搜索**：`Cmd+K`（Mac）或 `Ctrl+K`（Windows）打开全文搜索
- **顶部右上**：中英文切换
- **底部翻页**：上一节 / 下一节

<TipCallout type="warn" title="关于版本">
  本书基于 n8n 1.x 版本编写。n8n 迭代很快，遇到 UI 细微差异时，**核心概念依然适用**，照着调即可。每节末尾会标注最后核对日期。
</TipCallout>
